# Abyss Outer Rim v001

Managed generated sources for the fixed map `ABYSS_FIELD`.

- wall/floor masters: dedicated ominous ritual-ruin terrain.
- chasm: bottomless central entrance, used as an impassable adjacent action.
- altar/pillar: manually authored blocking props; never random floor decoration.
- `tools/assets/build-abyss-outer-assets.py` creates registered terrain variants and transparent map overlays.

The map keeps a tested route from the entry point to a tile adjacent to the chasm.
