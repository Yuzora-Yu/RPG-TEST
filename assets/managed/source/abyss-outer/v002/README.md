# Abyss Outer Rim v002

The fixed outer-rim map now uses two intentionally different ruin-floor families:

- `ritual-floor`: restrained concentric ritual slabs for the inner route.
- `prism-paving`: broken purple flagstones with old bronze channels and empty prism-socket traces for the outer court.

The prop set contains independent silhouettes rather than mirrored copies: intact columns, a half-collapsed stump, a fallen pillar, a ruined arch, an altar, and two different empty prism pedestals. Six pedestal ruins are manually placed away from the central chasm while all four adjacent interaction cells remain open.

Build with `tools/assets/build-abyss-outer-v002-assets.py`.
