# 光の翼・主人公歩行チップ v002

通常の主人公歩行チップを変更せず、方向別に画像生成した翼レイヤーを背面合成する。

- `hero-wing-layer-*-chroma.png`: 組み込み画像生成で作成した緑背景の原画
- `hero-wing-layer-*-alpha.png`: クロマキー除去後の方向別マスター
- `hero-wing-v002-preview.png`: ゲーム用8フレームの3倍表示確認シート
- ゲーム用出力: `assets/generated/hero-wing-{direction}-{step}-v002.png`
- 再構築: `tools/assets/build-hero-wing-sprites.py`

方向ごとの原則:

- `down`: 正面から見て両翼は身体・武器より後ろ。歩行用にやや畳む。
- `up`: 翼の裏面を見せ、マントの後ろから左右へ出す。
- `left`: 付け根は背中側、羽先は画面右へ流す。遠い翼は細く見せる。
- `right`: 付け根は背中側、羽先は画面左へ流す。遠い翼は細く見せる。

生成指定は、SFC後期からPS1初期相当の制限色ドット絵、暖色寄りの象牙色と淡い金色、過剰な発光・光粒・巨大な戦闘用展開・写実的な羽毛・AI的な微細テクスチャを避ける方針で統一した。
