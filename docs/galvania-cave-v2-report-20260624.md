# ガルヴァニアへの洞窟 v2 全面刷新レポート

## 設計修正

- `GALVANIA_CAVE_NORTH` / `GALVANIA_CAVE_SOUTH` の2ダンジョン構成を廃止し、`GALVANIA_CAVE` 1本に統合。
- ワールド側は1つのエリアに `north` / `south` の2入口を持たせる方式に変更。
- 北口は1階開始、南口は6階開始。
- 北口は従来通り `lightPalaceCleared` を要求。
- 南口は、6階南口を内側から出たときに `galvaniaCaveSouthOpened` を立て、以後のみ再進入可能。

## MAP刷新方針

ロンダルキアへの洞窟系の「正解ルートを覚える高難度」と、ネクロゴンドへの洞窟系の「火山・溶岩を越える通過儀礼」を意識し、通り道でありながら密度の高い6階層に再設計した。

- 1階: 北口の黒岩洞。複数ループと溶岩噴気、倒れている兵。
- 2階: 無限回廊。石碑ヒント、複数ワープ、限定マップ表示。
- 3階: 溶岩地底湖。細い橋、溶岩ダメージ、任意ボス、限定マップ表示。
- 4階: 氷晶十字滑床。十字路で曲がれない氷床、金鍵任意ボス。
- 5階: 魔軍補給路。金鍵ショートカット、長い迂回路、徘徊魔族、限定マップ表示。
- 6階: 南口の白骨旧坑。南口開通フラグ、最後の惑わせ通路、任意宝箱ボス。

## 中ボス方針

必須戦闘は置かず、すべて任意。役割は以下に限定。

- レア宝箱の守護
- 金鍵によるショートカット解放
- 探索上のリスク・報酬選択

## 追加ロジック

- 固定ダンジョンの入場判定を入口キー対応に変更。
- 固定ダンジョン出口リンクに `setFlag` / `setFlags` を追加。
- 汎用 `entryRequiredFlag` に `entryRequiredEntrances` を追加し、北口だけ従来フラグを適用可能にした。
