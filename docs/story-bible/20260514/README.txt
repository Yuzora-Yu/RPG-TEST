# PRISMA ABYSS ストーリー・キャラクター設定整理パック

作成日: 2026-05-14

このパックは、現在の `characters.js` のキャラクター設定、既存の `story.js` の序盤実装、今後のストーリー開放方針、六元素・プリズマ・深淵の世界観メモを統合して整理したものです。

## 収録ファイル

- `01_内部用_世界観設定集.md`
- `02_内部用_ストーリー進行と機能解放方針.md`
- `03_内部用_キャラクター設定集_by_area.md`
- `04_内部用_キャラクター相関整理.md`
- `05_内部用_隠し設定と今後の伏線.md`
- `06_ゲーム内用_オープニングプロローグ案.md`
- `07_Codex向け_実装指示メモ.md`
- `08_characters_current_data_summary.md`
- `09_characters_current_data_summary.tsv`
- `10_編集メモ_今後characters_jsへ入れたい項目.md`

## 整理方針

この資料では、ゲームの核を「ストーリーを進めることで、仲間・施設・移動手段・やり込み要素が少しずつ解放されるRPG」として定義しています。

特に重要なのは、ガチャや深淵周回を序盤から開放するのではなく、本編中はストーリー加入キャラを中心に進め、魔王城攻略後から本格的なやり込みへ移行する構造です。

## 注意

一部、現行の `characters.js` アーカイブ文と既存 `story.js` の序盤描写には調整余地があります。たとえば、アルスの「滅びた故郷」と、ゲーム開始地点である「始まりの村」は同一にしない方が整合しやすいです。このパックでは「アルスは別の滅びた故郷の出身で、始まりの村は最初に救う村」という整理を推奨しています。
