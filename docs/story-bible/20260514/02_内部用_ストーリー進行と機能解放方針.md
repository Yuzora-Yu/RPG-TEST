# 02_内部用_ストーリー進行と機能解放方針

## 基本思想

このゲームは、序盤から全機能を開放するゲームではなく、ストーリーを進めることで世界とシステムが広がるRPGにする。

本編の中心はレベル50程度までのストーリー攻略。深淵周回、転生、ガチャ、属性エンドレス探索は、本編後半からクリア後のやり込みに寄せる。

## 推奨進行順

```text
始まりの村
↓
炎の里
↓
風の集落
↓
水上都市
↓
雷の要塞
↓
光の宮殿
↓
魔王城
↓
深淵の魔窟
```

六元素の攻略順は **火 → 風 → 水 → 雷 → 光 → 闇**。

## ストーリー加入と機能解放

| 進行 | 主な加入・登場 | 解放したい機能 | 備考 |
|---|---|---|---|
| 始まりの村 | ガイル、サラ | 基本操作、戦闘、道具 | アルスは外から来た者として整理すると、滅びた故郷設定と整合する |
| 炎の里 | シャオ、カリン | 鍛冶屋 | 火＝鍛造。シャオとシャニーの伏線を初めて匂わせる |
| 風の集落 | エリーゼ、アリサ、ハイネ、マリー、リーシアの噂 | 情報・噂・状態異常チュートリアル | リーシアは直接登場させず、噂だけで十分 |
| 水上都市 | ケイト、ソフィア、シルビア、アラン、ハヤテ | 船、カジノ | 水＝記憶。アラン/ハヤテ、ケイト/ソフィア、ジョセフ/シルビアの過去を断片的に扱う |
| 雷の要塞 | ジョセフ、フリーダ、バロン、リン、ゼリード | メダル王 | 雷＝契約・功績。メダル王は功績を認める施設にする |
| 光の宮殿 | レイラ、クロード、レオン、ルーナ、リュシオン念話 | 深淵・転生の伏線 | 光＝秩序。レオン/クロード/レイラ/ルーナの関係を整理する |
| 魔王城 | シャニー、ゼノン、ミネルバ、リュウ | ガチャ、本編終盤 | ゼノンは悪役であり続けるが、真意は最後まで伏せすぎない |
| 深淵の魔窟 | 全員の因縁が集まる | 深淵、転送、転生、翼、エンドレス探索 | 40階で表エンド、100階で翼、150階以降は転生前提 |

## ガチャの扱い

本編中はガチャを解放しない方針を推奨。

理由:

- ストーリー加入キャラの存在感を守るため
- 序盤の難易度とキャラ導線を壊さないため
- 魔王城攻略後に「歴戦の英雄・深淵の縁者・神域級キャラを召喚できる」構造にすると、解放時の納得感があるため

魔王討伐後にガチャ解放。これにより、SSR/UR/EXキャラは本編中にNPC・助っ人・噂・イベントで顔見せし、正式な恒常入手はクリア後に回せる。

## 深淵の段階

| 段階 | 内容 |
|---|---|
| 初探索 | ダンジョン突入機能を解放。チュートリアルを入れる |
| 40階ボス討伐 | 表向きのエンディング。光の宮殿イベント発生。転生の実を1つ配布 |
| 光の宮殿イベント後 | 41階以降、宿屋転送、属性エンドレス探索を解放 |
| 100階攻略 | 光の翼を解放 |
| 150階以降 | 転生前提のボス難易度へ |

| 光の宮殿 | 25〜30 | 光多め、光耐性高め、闇に弱い |
| 魔王城 | 30〜40 | 闇多め、闇耐性高め、光に弱い |
