# 06_ゲーム内用_オープニングプロローグ案

## 方針

ゲーム内のプロローグは、設定を説明しすぎない。プレイヤーが最初に知るべきことは以下だけ。

- 世界には六元素がある。
- プリズマという結晶がある。
- かつて光が割れ、深淵が生まれた。
- アルスはすでに何かを失っている。
- 物語は始まりの村から動き出す。

魔王の真相、リュシオン、王国儀式の詳細は後でよい。

## オープニング標準版

```text
かつて、この大陸には
六つの力が巡っていた。

火は、命を燃やし、鉄を打った。
風は、声を運び、道を開いた。
水は、傷を癒やし、記憶を沈めた。
雷は、意志を刻み、契約を結んだ。
光は、秩序を照らし、祝福を与えた。
闇は、終わりを受け入れ、眠りを守った。

六つの力は、争うものではなかった。
互いに欠けたものを補い、
世界をひとつの輪として保っていた。

人々は、その輪の中で生きていた。

山を削り、畑を耕し、
川を渡り、火を囲み、
祈り、眠り、また朝を迎えた。

その暮らしのそばには、
いつも虹色の結晶があった。

プリズマ。

六つの力を映し、
曲げ、整える結晶。

王国はそれを掘り出し、
教会はそれを祝福し、
賢者はそれを読み解いた。

プリズマは、繁栄の光だった。

だが、人はやがて願った。

巡る力を、ただ受け入れるのではなく、
自らの手で完全に支配したいと。

衰えぬ王国を。
尽きぬ祝福を。
揺らがぬ秩序を。
完全な光を。

そして、王都で大いなる儀式が行われた。

記録には、こうある。
六つの力を調停し、
世界にさらなる祝福を与えるための儀式だった、と。

けれど、その日、
光は白へ戻らなかった。

光は、割れた。

割れた光は七色に散り、
七色は、色ではない何かへ変わった。

火は熱を失わずに暴れ、
風は声を運ばずにざわめき、
水は記憶を濁らせた。

雷は鳴らずに契約を焼き、
光は祝福ではなく裁きとなり、
闇は眠りではなく底なしの沈黙となった。

そして大地に、深さが生まれた。

それは洞窟ではなかった。
鉱山でもなかった。

地図の下に開いた穴ではなく、
世界の内側に走った裂け目。

人々は、それを深淵と呼んだ。

王国は揺らぎ、
教会は沈黙し、
各地の拠点は、それぞれの元素を抱えて孤立していった。

炎の里。
風の集落。
水上都市。
雷の要塞。
光の宮殿。
そして、北方にそびえる魔王城。

魔王は世界を壊す者だと、人々は語る。

けれど古い伝承には、こうも記されている。

魔王とは、
世界が壊れぬよう、
壊れ役を引き受けた者である、と。

真実は、まだ誰も知らない。

始まりは、小さな村だった。

大陸の片隅。
王国の光も、
魔王城の影も、
まだ遠い場所。

そこへ、ひとりの若者が流れ着く。

名は、アルス。

すでに故郷を失い、
すでに祈りの答えを失い、
それでも剣を手放せなかった者。

夜明け前。
村には、不吉な気配が満ちていた。

井戸の水は淡く濁り、
森の鳥は一羽も鳴かず、
村はずれの祠では、
誰もいないはずの奥から
鉄を打つような音が聞こえていた。

そして、子どもの悲鳴が上がる。

深淵は、遠い場所の話ではなかった。

これは、六つの元素を巡る旅。

これは、失われた均衡を取り戻す物語。

そしてこれは、
深淵の底に落ちた世界の本音を、
まだ何も知らない者たちが
拾い上げていく物語である。
```

## ゲーム開始直後用の短縮版

```text
かつて、この大陸には
六つの力が巡っていた。

火、風、水、雷、光、闇。

それらは魔法ではなく、
世界そのものを動かす法則だった。

人々は虹色の結晶、
プリズマの光を借りて暮らしていた。

王国は栄え、
教会は祈り、
賢者は元素の理を読み解いた。

だが、王国はやがて願った。

巡る力を受け入れるのではなく、
人の手で完全に支配したいと。

そして、王都で大いなる儀式が行われた。

その日、光は割れた。

六つの力の隙間から、
存在しないはずの歪みがあふれ出した。

人々はそれを、混沌と呼んだ。

大地には、深淵が開いた。

それは洞窟ではない。
世界の内側に生まれた裂け目だった。

王国は揺らぎ、
各地の拠点は孤立し、
北方の古城は魔王城と呼ばれるようになった。

けれど、真実はまだ誰も知らない。

魔王は本当に世界の敵なのか。
神はなぜ沈黙したのか。
深淵の底には何があるのか。

始まりは、小さな村。

すでに故郷を失った若者、アルスが、
新たな悲鳴に剣を取るところから、
止まっていた物語は動き出す。
```

## 既存 `GAME_START_1` へ接続する導入文

```text
――始まりの村。

大陸の片隅にある、小さな村。

王国の都からは遠く、
魔王城の噂も、
深淵の恐怖も、
まだどこか遠い話だった。

けれど最近、
村の近くの森で魔物が増えはじめている。

井戸の水は、夜になると淡く光り、
古い祠では、誰もいないはずの奥から
金属を打つような音が聞こえるという。

夜明け前。
村には、不吉な気配が満ちていた。
```

この後、現行の `GAME_START_1` の「子供が魔物に襲われている！」へ接続すると自然。
