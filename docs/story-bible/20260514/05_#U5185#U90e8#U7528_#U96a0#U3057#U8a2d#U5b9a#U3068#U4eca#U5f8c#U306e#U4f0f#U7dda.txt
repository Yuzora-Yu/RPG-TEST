# 05_内部用_隠し設定と今後の伏線

## 1. アルスの故郷問題

現行 `characters.js` のアルスは「滅びた故郷の復讐者」として設定されている。一方、現行 `story.js` では始まりの村から物語が開始しており、ガイルとサラはその村の住人として描かれている。

推奨整理:

- アルスの本当の故郷は、始まりの村とは別にすでに滅びている。
- アルスは深淵の魔物を追って旅をしており、その途中で始まりの村を救う。
- 始まりの村は、アルスが「復讐者」から「仲間を持つ旅人」へ変わる最初の場所。

これにより、アルスの重いアーカイブと、序盤のガイル/サラ加入が両立する。

## 2. 深淵の王

アルスのアーカイブでは、故郷を奪った元凶として「深淵の王」が示唆されている。

この存在は、現行の敵ID `4001 深淵なる闇` と統合してよい。ゼノンとは別物にすること。ゼノンは魔王だが、深淵の王ではない。

推奨構造:

- 民衆が恐れる魔王: ゼノン
- 本当に世界を壊す元凶: 深淵の王/深淵なる闇
- 序盤の誤解: 「魔王が深淵を開いた」
- 中盤の反転: 「魔王は深淵を抑えていた」
- 終盤の真相: 「王国の儀式が混沌を生み、深淵の王を呼んだ」

## 3. シャオとシャニー

ほぼ確定で姉妹関係として整理するのがよい。

シャオ側の認識:

- 姉は魔王の下へ行った。
- 姉は家族と里を裏切った。
- 自分は震えて見送るしかなかった。
- だから髪を断ち、女としての弱さも、妹としての甘えも捨てた。

シャニー側の真相:

- 自身の瘴気が妹や故郷を蝕むと知った。
- ゼノンの制御術と引き換えに、才能と生涯を差し出した。
- 妹が生きる世界を買うために、裏切り者として消えた。

魔王城でこの真相を出すと非常に強い。

## 4. 魔王ゼノンの扱い

ゼノンは「実はいい人」にしすぎない方がよい。

彼は冷酷で、苛烈で、恐怖を使う。ただし目的は世界の破壊ではなく、混沌から世界を保つこと。

プレイヤーがゼノンを倒すべき敵として進んできた後、ゼノンが深淵と戦っていたことを知る構成がよい。ただし、ゼノン自身は言い訳をしない。悪として憎まれる役を引き受けているため。

## 5. 光の宮殿の裏テーマ

光の宮殿は「光＝善」の場所ではなく、「光＝秩序、祝福、裁き」の場所。

- レオン: 温かい正義を持つ完成形に近い盾。
- レイラ: 正義が硬すぎる若い騎士。
- クロード: 闇に堕ちかけ、救われた光の武人。
- ルーナ: 神によって戻された聖女。奇跡そのものだが、奇跡の危うさも背負う。
- リュシオン: 神でありながら、人を愛しすぎて均衡を破った存在。

深淵40階後のイベントは、光の宮殿で行うのが最適。転生の実は「やり直し」ではなく、「終わった命を、別の形で巡らせる」光と水の混合儀礼として扱える。

## 6. ジョセフとレオンの親子関係

**確定裏設定：ジョセフはレオンの父。**

この設定はすぐに表へ出さず、雷の要塞・水上都市・光の宮殿をまたぐ長い伏線として扱う。

- ジョセフは、かつて王国軍の精鋭であり、民を守る盾だった。
- レオンは、その背中を見て育ったが、父の沈黙と責任の取り方を許せなかった。
- レオンの「温かい正義」は、ジョセフのように黙って背負う正義への反発から生まれた。
- ジョセフはレオンを愛していないわけではない。ただ、自分は父と名乗る資格を失ったと思っている。
- レオンはジョセフを嫌悪しているが、完全には捨てられない。畏怖、恨み、感謝、憧れの残骸が混ざっている。

演出案:

- 雷の要塞では、ジョセフが「白銀の騎士」の名を聞いてわずかに沈黙する。
- バロンは事情を知っているが、本人たちが語るまで口を出さない。
- シルビアは「あの日」の記憶を通じて、ジョセフが何を失ったかを知っている。
- 光の宮殿でレオンが登場した時、ジョセフを知るNPCが「よく似ている」と漏らす。
- 親子関係の明示は、光の宮殿終盤または魔王城前の重い会話に回す。

この関係により、雷の要塞のテーマである「契約・責任・裁き」と、光の宮殿のテーマである「秩序・祝福・正義」が親子関係でつながる。

## 7. ジョセフ、バロン、シルビアの「あの日」

現行アーカイブでは、ジョセフとシルビアの間に語られぬ「あの日」がある。バロンも同じ時代にジョセフと肩を並べた戦士。ここは、ジョセフがレオンの父であることを裏から補強する過去事件として扱う。

推奨設定:

- かつて王国軍と傭兵団が共同で守った村/戦線があった。
- ジョセフは王国軍として、バロンは傭兵団として参戦。
- シルビアはその地で公演/慰問/祝祭を行っていた、または戦意高揚の象徴だった。
- 防衛は成功したが、何か一人または一隊を見捨てる決断があった。
- ジョセフは責任を背負い、シルビアは愛を知らない舞台人として凍った。

この事件を雷の要塞か水上都市の回想で扱うとよい。

## 8. マリーとミネルバ

この二人は、宗教と科学の対比に使える。

- マリー: 祈りは届かなかった。だから光を拒絶するために祈る。
- ミネルバ: 祈りでは救えない。だから証明し、解析し、禁忌に踏み込む。

ただし、どちらも「人を救いたかった」点では同じ。対立させるより、互いに相手の選択を痛みとして理解している方が深い。

## 9. リーシアの出し方

リーシアはSSRであり、設定上も伝説級。序盤から直接同行させると世界観が壊れやすい。

推奨:

- 風の集落では噂のみ。
- ソフィアが名前を出して対抗心を見せる。
- リュウやアランの過去に「魔道の基礎を授けた者」として登場。
- 本格登場は深淵中盤以降、または魔王後ガチャ/イベント。

## 10. 住民NPCと会話への落とし込み

固有MAPに住民を配置する際、各地の会話で伏線を薄く入れる。

例:

- 炎の里の老人: 「あの忍の娘は、里を捨てたのではない。捨てられたのは、むしろあの子の方かもしれん」
- 風の集落の旅人: 「リーシア様？ 風の音を聞け。名を呼ぶと、どこかで星が落ちるぞ」
- 水上都市の劇場係: 「シルビア様は舞台では笑う。でも幕が下りると、いつも北の方を見ている」
- 雷の要塞の兵士: 「ジョセフ殿とバロン殿が同じ酒場にいる時は、誰も昔話をしない」
- 光の宮殿の巫女: 「神は沈黙したのではありません。声を出すための喉を、世界が失ったのです」
- 魔王城の魔族: 「我らが王は残酷だ。だが、約束だけは破らぬ」

## 11. 今後 `characters.js` に追加したいメタ情報

アーカイブ本文に関係性が埋もれているため、今後のイベント実装を楽にするなら、キャラごとに以下のような項目を追加したい。

```js
story: {
  originArea: "FIRE_VILLAGE",
  joinTiming: "after_fire_village_clear",
  storyRole: "story_join",
  relationshipTags: [
    { target: 306, type: "sister", feeling: "hatred_misunderstanding" },
    { target: 402, type: "enemy", feeling: "distrust" }
  ],
  hiddenFlags: ["shao_shanny_sisters"],
  spoilerLevel: 2
}
```

これを入れると、NPC会話、加入イベント、実績、図鑑、ガチャ演出、チュートリアルが作りやすくなる。
