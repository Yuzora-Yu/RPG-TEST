# ランダムダンジョン「森100%」外観検証版

## 目的

床・壁・戦闘背景のランダム外観が、抽選後や再開時に深淵外観へ上書きされないことを実機プレイで確認するための一時設定。

## 検証結果

2026-07-15、実機で禁忌の森の床・壁・戦闘背景への切替を確認済み。二重初期化による深淵外観への上書き原因を除去し、通常のランダム抽選へ復帰した。
検証版の識別情報が保存された現在階は、通常版での初回復元時に外観を一度だけ再抽選する。

## 検証時の設定

- `Dungeon.randomVisualThemeTestOverrideId` を一時的に `forbidden-forest` に設定した。現行値は `null`。
- 10階ごとのボス階を除くランダムダンジョンは、抽選値にかかわらず禁忌の森外観を100%使用する。
- 新規生成階では床・壁・戦闘背景をすべて禁忌の森へ統一する。
- 保存済みの非ボス階を再開した場合も、保存されていた本来の抽選結果を壊さず、表示だけ禁忌の森へ切り替える。
- 検証対象階の表示名には `【森固定検証】` を付ける。
- ボス階および深淵系外観では、従来の壁面・松明表示を維持する。
- 北東の洞穴は `useDungeonWallFace: true` により、壁面・松明表示を維持する。

## 再検証時

必要な場合だけ `Dungeon.randomVisualThemeTestOverrideId` にテーマIDを設定する。通常の階層別候補と確率抽選、保存・再開時のテーマ復元ロジックは削除しない。
