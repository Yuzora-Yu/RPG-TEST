## 火の里

1. 里の長から相談を受ける。
   炎が弱まり武器を作れなくなった。火のプリズムが納められているイグナ火山に魔物が出るようになり、里の者では太刀打ちできない。何人も帰ってこないため、様子を見てきてほしいと頼まれる。

2. 周囲の村人から情報を得られる。
   「武器が作れなくなったらこの里は終わりだ」「王国軍が視察に来てから火が弱くなったような気がする」と聞ける。

3. 村マップ北に火山入口を設置する。
   火山入口直前でシャオが話しかけてくる。生まれ育った里のために火山へ入ったが、一人では奥まで行けず困っていた。同行許可で仲間になる。断ると「そんなこと言わないで、頼む」と再度頼まれ、許可までループする。

4. イグナ火山ダンジョン。
   火山の奥で、火のプリズムの周囲に王国軍らしき兵士を発見する。問いただそうとすると侵入者扱いされ、ブロンズリーダー、ブロンズナイト3体と戦闘。

5. 炎楔のグラド戦。
   戦闘後、プリズムに手を当てていた長身の男が振り返り「少し遅かったようだな」と告げる。プリズムの光がさらに弱まり、男から炎のオーラがあふれる。「我、炎獄の秘奥を得たり。この国の未来のため、灰となれ、冒険者よ」で戦闘開始。

6. グラド勝利後。
   炎の力がプリズムへ戻る。グラドは闇の中へ撤退する。主人公たちは王国軍が闇の力を使っていることに違和感を覚え、この国が悪い方向へ進んでいるのではと危惧する。
   グラド戦に負けた場合はダンジョン奥で再開し、「何度来ても無駄なこと…炎の力を吸い尽くすまで、邪魔はさせぬぞ！」に会話が変わる。

7. 里の長へ報告。
   炎の力が戻り鍛冶が可能になる。原因を伝え、王国へ向かうことを話す。長から、あの日から世界の理がずれて船なしでは王国に行けなくなったと聞き、まず北の風の集落へ向かうよう提案される。シャオは王国軍が闇の力を使ったことに強い不快感を示し、正式同行を申し出る。ここで鍛冶機能を解放し、チュートリアルを差し込む。

## 風の集落

1. 集落には子どもしかいない。
   最年長のエリーゼに話しかけると、大人たちはいつのまにか姿を消したと聞く。目撃した子どもは、森に飲み込まれていったと話す。

2. 禁忌の森へ。
   火の里と同じような異変の可能性を考え、集落西から禁忌の森へ向かう。かつて聖域だったが、プリズム崩壊の日から魔物が溢れ、風が淀み、封じられた場所。案内役としてエリーゼが同行する。

3. 祈りの広場。
   森中央の石碑を調べると不穏な風を感じ、シルフウルフ、ヒールフェアリーと戦闘。

4. 神獣の浄化。
   勝利後、火のプリズムのオーラを受けて呪いから浄化された2体から話を聞く。風のプリズムを守る神獣だったが、呪われて行動がおかしくなっていた。風のプリズムの力が弱っているかもしれない。大人たちも風の神殿へ向かっていた。ヒールフェアリーがHPとMPを全回復してくれる。

5. 風の神殿。
   最奥の祭壇には、光が弱まった風のプリズムとうつろな目の大人たちがいる。風のプリズムの上に座り、崇められながら力を奪う「風楔のエリシア」と戦闘。

6. 勝利後。
   エリシアは闇の中へ逃れる。集落に聖なる風が戻る。集落の長へ王国へ行く船を探していることを伝えると、西の水上都市を勧められる。エリーゼは世界に不幸が増えることを憂い、正式同行を申し出る。

## 水上都市

1. 街は寂れ、黒い鎧の兵士が我が物顔で歩いている。

2. 街の西で、兵士に怒鳴られて連れていかれそうな少女を発見。話しかけるとダークソルジャー2体と戦闘。

3. 勝利後、宿の酒場へ案内される。
   妖艶な賢者ソフィアから、水が濁り、王国兵の休息地にされている現状を聞く。ソフィアは海底神殿の調査に来ていたが、不審視され捕まると親戚の宿にも迷惑がかかる。

4. ケイト同行。
   火・風のプリズムで起きたことを伝え、王国軍の関与が決定的になる。水のプリズムがある海底神殿へ向かうことに。ソフィアは動けないため、弟子のケイトを同行させる。

5. 海底神殿入口。
   水上都市北にある。入口は兵士に封鎖され、話しかけるとダークナイト、ダークソルジャー2体と戦闘。

6. 海底神殿。
   雑魚として魔物だけでなくダークナイト、ダークソルジャーも頻出。各フロアで鍵扉が多く、ダークナイト戦で鍵を得て進む構造。水で区切られたフロアを多めにする。

7. 最奥の祈祷の間。
   水のプリズムから力を奪い、フロア全体を凍らせている氷槍使い「氷楔のシーリス」およびダークナイト2体と戦闘。戦闘直前に全体を冷気で凍えさせる演出を入れ、戦闘開始時は確定不意打ち。

8. 勝利後。
   水のプリズムに力が戻り、水の流れが正常化する。水上都市から兵士が消え、活気が戻る。都市の長から船を得る。翼取得になっている既存コードは修正する。

9. ソフィアとケイト。
   ソフィアは楔たちの言動を詳しく調査すると話す。ケイトについて「少しいい顔になった」「足りなかったのは実戦」と語り、ケイトが正式加入する。

10. 次目的。
   船で川を進めば、魔王城と光の神殿の間を守る雷の要塞へ行ける。街の西の島にはメダル交換所がある。地殻変動で島になった。

## 雷の要塞

1. 到着するとジョセフに止められる。
   警戒するが味方だった。

2. 要塞異変。
   雷のプリズムで機械を動かし魔王軍の侵攻を防いでいたが、機械が暴走し入れなくなっている。プリズムが狙われていると判断し、強行突入する。ジョセフも「子供だけにやらせちゃ、騎士の名がすたるぜ」と同行する。

3. 要塞内部。
   バトルマシーン、サンダーアーマーとの戦闘が要所で発生。倒すと鍵を得る。中ボス撃破で鍵を得て進む構造にする。雷ダメージ床も配置したい。

4. 雷楔のレナード。
   最奥に雷をまとう剣士レナードが登場。高潔で誠実だが国のため従う。ジョセフと口論後、戦闘。

5. レナード勝利後。
   「プリズムの力は必要な分は既に光の神殿に集まっている…儀式の日は近い。止めたいのならば、まずは大灯台で発動している魔術結界を壊すことだな」とこぼす。「それから…」と言いかけたところで聖騎士ヴェルドが現れ、レナードを斬る。レナードは闇の中へ消える。

6. 聖騎士ヴェルド戦。
   「お前たちは救世の障害だ。大いなる祝福には、犠牲が伴うもの。犠牲無しに、世界を変えることなどできない。」「お前たちは世界を救っている気になっているようだが、ゆるやかな死に向かっているだけよ。死ぬのなら、貴様たちだけで死ね。」と語り、戦闘。負けイベント。絶対に倒せないHPと高火力。倒してもイベント後に強制全滅会話。

## 大灯台

1. 中階層でヘルクラッシャー、デモンキャットと戦闘。鍵を取得。

2. 最上階で常闇のリリスと戦闘。

3. 勝利後、光の神殿の結界が消え、入口から奥へ入れるようになる。

## 光の神殿

1. 大灯台未攻略なら入口に結界があり奥へ進めない。

2. 最奥で魔道神官ジャスパー、聖騎士ヴェルドと戦闘。

3. 一度負けイベントで全滅するが、リュシオンの加護により光のプリズムが復活。ジャスパー、ヴェルドの加護が消え弱体化する。

4. 勝利後、レイラが加入する。ジャスパーとヴェルドはいつの間にか消えている。

5. 残るプリズムは闇。西の果ての魔王城にある。討伐に向かう。

## 魔王城

1. 中ボスとして常闇のゼルドラス、冥騎士ベレト、風詠のエルメナスが登場。

2. 最奥で魔王ゼノンと戦闘。

3. 闇のプリズムは適切に守護され、力を奪われていなかった。魔王は王国を攻めていたのではなく、王国軍から守り続けていた。

4. 真の問題は、あの事件以降生まれた混沌の力。深淵から漏れ続ける闇より深い混沌こそが元凶であり、王国も混沌に魅入られたのだろうと語る。

5. 通常なら深淵に呑まれるだけだが、6属性の力に触れ加護を得た主人公なら入れるかもしれない。魔王城より南東、世界の中心に深淵への亀裂があると教えられる。

6. 戦闘勝利でシャニーが加入。

## 深淵の入口

6属性の加護を得た状態で中央のひずみに来るとイベント発生。ダンジョンが解禁され、以降挑めるようになる。
