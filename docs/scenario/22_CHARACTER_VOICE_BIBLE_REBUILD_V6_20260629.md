## シャオ

### 基本

- charId: 105
- 一人称: 私。強く動揺すると「僕」が漏れる可能性あり。ただし本編序盤では乱用しない。
- 口調: 礼節がある。怒ると短くなる。
- 根: 姉シャニーへの怒り、里を守れなかった無力感、強くあろうとする意地。

### 誤解

シャオは、シャニーが里を捨てて魔王軍へ入ったと思っている。
そのため魔王軍への敵意が強い。

ただし、本当はシャニーが混沌に汚染され、ゼノンに救われたことを知らない。

### 好きなもの

- 朝の鍛錬後に飲む水。
- 短く整えた髪。昔の自分を断つためでもある。
- 約束を守る人。
- 手入れされた訓練場。

### 嫌いなもの

- 裏切り。
- 曖昧な返事。
- 弱い自分を見られること。
- 高い場所で足元を見ること。

### 台詞の方向

怒りはまっすぐ。
ただし、姉に対しては怒りだけでなく、置いていかれた痛みがある。

```text
そんな一言で……！
```

のように、言い返したいのに言葉が詰まる場面を作れる。

## シャニー

### 基本

- charId: 306
- 一人称: 私。
- 口調: 短い。必要なことだけ言う。
- 根: 幼い頃に深淵へ魅入られた少女。妹と里を呪わないため、生きるため、ゼノンとの契約を選んだ。自分が許されると思っていない。

### 正本背景

前版の「妹と里を守るために消えた」「里を守るために戻れなかった」という要約は不正確。
シャニーを綺麗な自己犠牲だけに寄せてしまい、キャラの痛みと契約の重さを削るため、以後使わない。

幼い頃、シャニーは深淵に魅入られた。
その身から溢れる瘴気は、愛する妹シャオや故郷の里を蝕み、いずれ滅ぼしかねない呪いだった。

絶望の中で、シャニーは魔王ゼノンに縋った。
ゼノンは彼女を救ったが、ただの慈悲ではない。
シャニーの才能と利用価値を見出し、自らを制御する術と引き換えに、彼女の才と生涯を契約で縛った。

シャニーはゼノンに感謝している。
同時に、その契約が優しいだけの救いではなかったことも知っている。
魔王軍に与したのは、里を嫌ったからではない。
汚染されたまま故郷へ戻れば、自分が妹と里を壊す側になっていたからである。

### 好きなもの

- 古い詩集。声に出して読むより、指で文字を追う方が落ち着く。
- 夜明け前の静けさ。誰も自分を見ない時間。
- 妹が昔くれた小さな飾り。持っているかは未確定。
- 刃物の手入れ。汚れを落とせる作業だから。

### 嫌いなもの

- 自分の瘴気の匂い。
- 妹の泣き顔。
- 優しい言葉。受け取る資格がないと思っている。
- 「裏切り者」と呼ばれること。否定したいが、何も説明できないから。
- 恩を美談にされること。ゼノンとの契約は、それほど綺麗なものではない。

### 台詞の方向

言葉は少ない。
謝罪を急がない。言い訳もしない。
妹に対しては、説明より先に「怒っていい」と受ける。
ただし、魔王城時点では正式な和解を行わない。

```text
里を嫌いになった日はない。
```

```text
うん。だから、怒ってていい。
```

```text
生きるために、選んだ。
あなたのそばには、戻れなかった。
```

## 魔王ゼノン

### 基本

- charId: 402
- 一人称: 余 / 俺は場面で調整。魔王としては余。
- 口調: 尊大。だが約束に厳しい。
- 根: 魔王の名を背負い、闇と混沌を監視してきた孤独な守護者。

### シャニーとの関係

ゼノンはシャニーを救った。
ただし、それは慈悲だけではない。
シャニーに縋られ、彼女の才能と利用価値を見出し、制御の術を与える代わりに、才と生涯を契約で縛った。

ゼノンは善人として扱わない。
だが、契約した者を雑に捨てる者でもない。
救い主、契約者、支配者が同居している。

シャオへ真実を全部説明する役ではない。
少し言いかけ、シャニーが止めることで、関係の深さと未解決感を出す。

### 好きなもの

- 約束を違えない者。
- 恐れても立つ者。
- 闇を闇として見られる者。
- 玉座の静けさ。

### 嫌いなもの

- 善人扱い。
- 約束を破る者。
- 光を名乗る支配。
- 深淵が闇を餌にすること。

### 台詞の方向

尊大だが、弱者を嘲る尊大さではない。
自分を善と呼ばれることを嫌う。

```text
余を善と呼ぶな。不快だ。
だが、預かったものは捨てぬ。
```

## ソフィア

### 基本

- charId: 202
- 一人称: 私。
- 口調: 余裕がある。少しからかう。だが、頼む時は頼む。
- 根: 天才賢者。ケイトの才能を見抜いている。自分が動くと周囲を巻き込む立場も理解している。

### 注意

読者に嫌われる乱暴な依頼をさせない。

NG:

```text
こういう時に突っ込むのが仕事だろ？
```

OK:

```text
巻き込んですまない。だが、あんたたちに頼ませてくれ。
```

### 好きなもの

- まだ誰も解いていない謎。
- よく乾いた紙と古いインク。
- 弟子の成長を本人より先に見ること。
- からかった相手の反応。

### 嫌いなもの

- 分かったふりをする凡才。
- 宿や街の人間を巻き込むこと。
- 自分の届かない場所で弟子が傷つくこと。

## ケイト

### 基本

- charId: 104
- 一人称: 僕。
- 口調: 丁寧で気弱。だが芯は負けず嫌い。
- 根: 師匠の影に隠れていたが、師すら見惚れる未完の才能がある。

### 好きなもの

- 静かな机。
- 乾いた本。
- 師匠に雑に褒められること。
- 誰にも見られない夜の練習。

### 嫌いなもの

- 人前で急かされること。
- 濡れた袖。
- 「どうせ師匠の弟子だから」と見られること。
- 怖くないふりをする自分。

### 台詞の方向

難しい造語を使わせない。
「術式の線」などは避ける。

```text
怖いです。正直、外に出るより、机で本を読んでる方がずっと楽です。
でも、街の水がこんな色になってるのに、僕だけ部屋に戻るのは……嫌です。
```

この方向。
