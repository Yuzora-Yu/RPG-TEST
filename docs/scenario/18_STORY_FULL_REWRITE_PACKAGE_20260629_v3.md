## 重要な正本更新

### シャニー / シャオ / ゼノン

前版の「妹と里を守るために戻れなかった人」という要約は誤り。
美談に寄せすぎており、シャニーの孤独、ゼノンの冷たさ、契約の代償を削っている。
この表現は以後使わない。

正しくは、以下。

シャニーは幼い頃、深淵に魅入られた。
その身から溢れる瘴気は、愛する妹シャオや炎の里を蝕み、滅ぼしかねない呪いだった。
戻りたかったかどうか以前に、汚染されたまま戻れば、彼女自身が故郷を壊す側になっていた。

絶望の淵で、シャニーは魔王ゼノンに縋った。
ゼノンはただ優しく救ったわけではない。
シャニーの才能と利用価値を見出し、自らを制御する術を与える代わりに、彼女の才と生涯を契約で縛り、配下に加えた。

それは「妹と里のための尊い自己犠牲」だけではない。
生きるため、妹を呪わないため、故郷を滅ぼさないために選ぶしかなかった、あまりに孤独な等価交換である。

シャオはそれを知らない。
だから彼女は、姉が里を捨てて魔王軍に入ったと思い、長く恨み続けている。

ただし、魔王城で正式な和解を行ってはいけない。
この時点で必要なのは、シャオの確信に小さなひびを入れること。
「自分が信じてきた真実に、誤りがあったのかもしれない」と思わせる一滴でよい。

### 魔王城の会話方針

ゼノンがすべてを説明し始めるのは不自然。
しかし、彼が少し言いかけ、シャニーが遮るのは自然。

この場面の目的は以下。

- シャオの怒りは消さない。
- シャニーの沈黙には理由があると示す。
- ゼノンは慈悲だけでシャニーを救ったわけではないと示す。
- ゼノンはシャニーの利用価値を見出し、代償を取った契約者であると示す。
- シャニーは妹と里を忘れていないが、自分の口でまだ語れないと示す。
- ただし、完全な和解は後の専用イベントへ残す。

### 炎の里

炎の里の異変は、「火が弱い」「消えた」だけではなく、**火が制御できない**ことを生活の困窮として描く。

例:

```text
こんな火じゃ全部真っ黒こげだ。
かと思えば、こっちは生焼け。まともな食事ができねえよ。
```

火は恵みであり、生活そのもの。
火が不安定になると、鍛冶だけでなく、飯、湯、灯り、子どもの世話、火傷、仕事の段取りが壊れる。

### 雷要塞

`THUNDER_LEONARD_CLEAR` は改悪しない。
既存の温度感、ジョセフの怒り、レナードの苦い現実感、ヴェルド配下の騎士の冷酷さ、「救世主ごっこは、ここまでだ。」の伏線性を維持する。

この場面は、説明削減の対象ではない。
むしろ、画面演出と台詞の噛み合わせがすでに成立している。

維持するべき要素:

- レナードはジョセフへ、まだ古い理想にしがみつくのかと問う。
- ジョセフは、痛みを引き受けることと奪うことは違うと返す。
- レナードは大灯台の魔術結界を示す。
- その先を話そうとしたところで騎士に斬られる。
- ジョセフが「レナード！！！」と叫ぶ。
- 騎士が「救世主ごっこは、ここまでだ。」と嘲る。
