## v3 policy

v2は好き嫌い・口調がまだ属性記号に寄りすぎ、また一部の台詞例が作り物臭かった。  
v3では、日常、生活、過去、他キャラとの距離からキャラを立てる。

- 好きなものは、属性ではなく生活に根差す。
- 嫌いなものは、性格や過去の傷に根差す。
- 口癖は記号化しすぎない。
- キャラを魅力的にする癖と、読者に嫌われる癖を分ける。
- 味方キャラは癖があっても、基本的に読者が嫌いにならないようにする。

## アルス

### Core

- 役割: 主人公、勇者、深淵への復讐から始まる旅人。
- 表情: 静か。感情が薄いのではなく、出し方を忘れている。
- 成長: 復讐の剣から、誰かを守る剣へ。

### Personality

- 必要以上に喋らない。
- 感情を説明しない。
- 頷き、視線、短い返答で意志を示す。
- 仲間が増えるほど、短い言葉の温度が変わる。

### Likes

- 焚き火のそばの沈黙。
- よく手入れされた剣。
- 誰かが普通に飯を食べている時間。
- 夜明け前の空。

### Dislikes

- 希望を押し付ける言葉。
- 失ったものを簡単に慰められること。
- 子どもの泣き声を放っておくこと。

### Speech

- 短い。
- 「……行こう」「それでいい」「助ける」など、行動に繋がる言葉が多い。
- 長い演説は基本的にしない。

### Do not write

- 主人公の意志を、別キャラに説明させない。
- 過剰な自己分析をさせない。

## Additional recurring checks

### 1. 好き嫌いを属性記号にしない

悪い例:

```text
火属性だから辛い料理が好き。
水属性だから雨が好き。
```

良い例:

```text
サラは薬草を干す匂いが好き。
ガイルは腹にたまる飯が好き。
ケイトは誰も急かさない時間が好き。
```

### 2. 口調の個性と不自然さを分ける

個性は必要。  
だが、読者が意味につまずく造語や抽象語は避ける。

### 3. 味方キャラを嫌われ者にしない

癖、皮肉、軽口はよい。  
しかし、味方を危険へ押し出して当然という言い方は避ける。

### 4. 重い場面ほど普通の言葉を混ぜる

強い場面では、詩的な台詞より普通の一言が刺さることがある。

```text
父さん……！
```

のような一言は、説明より強い。
