# PRISMA ABYSS 複数視点レビューと修正方針 2026-06-29

## 0. 今回の前提

この文書は、現行 `story.js`、`characters.js`、旧キャラ用.xlsx、これまでのシナリオ正本mdを読み直し、さらにユーザー指定の参考作品の「会話設計」を確認したうえで、PRISMA ABYSSのストーリー全体をどう磨くかを言語化する。

ここで目指すものは、綺麗な要約ではない。

- 中学生プレイヤーにも本筋が伝わること。
- 小説読者が人物の心を追えること。
- JRPGプレイヤーがボス前会話で熱くなれること。
- ゲーム慣れしていない大人にも、行動理由が自然に見えること。
- それでいて、伏線や対比を台詞で説明しすぎないこと。

そして何より、キャラの言葉が「設定」からではなく、「生活」「傷」「癖」「誰かとの関係」から出ていることを重視する。

## 1. 参照作品から拾うべき会話設計

※本文の模倣はしない。長文引用もしない。拾うのは構造と呼吸のみ。

### 1-1. 『無職転生』から拾うもの

見るべき点は、美文ではなく「情けなさと決意の同居」。

壮大な転生譚でありながら、入口はかなり俗っぽく、主人公の情けなさ・後悔・生活感が先に置かれる。そこから「今度こそ本気で生きる」という芯が立つ。PRISMA ABYSSのアルスも、最初から勇者然とした台詞だけを置くと薄くなる。復讐者でありながら、村の子どもを助け、薬草袋を受け取り、飯や眠りや震える手に触れて、少しずつ「守る勇者」になる方が強い。

PRISMAへの適用:

- アルスは無口でよいが、周囲の反応だけで聖人化しない。
- ガイルやサラが「勇者様」と呼ぶなら、村の伝承と実際に救われた体験の両方を置く。
- 序盤は重い運命より、薬草袋、鍋、夜明け、泥のついた靴などの生活物で感情を支える。

### 1-2. 『転生ごときで逃げられるとでも、兄さん？』から拾うもの

拾うべきは、日常会話の軽さから、急に狂気や危機へ滑り落ちる切り替え。

この作品は、普通の会話や食事・家族の場面に、ふと「話が通じない相手」の怖さが混ざる。敵の異常性を説明するより、敵本人が自然な顔でおかしなことを言うため、読者がぞっとする。

PRISMAへの適用:

- ジャスパーは「私は悪だ」と言わない。
- リリスは説明役になりすぎない。
- 歪んだ光、歪んだ救済は、本人が当然のように言うから怖い。
- 魔王軍誤認は、反論で弱めない。シャオが自然に誤認できる言葉を敵側に残す。

### 1-3. 『乙女ゲームのヒロインで最強サバイバル』から拾うもの

拾うべきは、役割を押し付けられた人物が「私は私だ」と生きる強さ。

この作品は、主人公がゲームのヒロインという役割から逃れ、自分の生存を選ぶ物語として提示されている。PRISMAでは、レイラ、ケイト、シャニーにこの構造が使える。

PRISMAへの適用:

- レイラは「王国の聖騎士」という役割から、自分で見て決める騎士へ移る。
- ケイトは「師匠の弟子」から、自分の魔術で誰かを助ける少年へ移る。
- シャニーは「ゼノンの道具」ではなく、妹と自分の罪を背負って歩く人間へ移る。

### 1-4. 『ソードアート・オンライン』から拾うもの

拾うべきは、ゲーム的設定を短い一文で命に接続する強さ。

「ゲームであっても遊びではない」のように、複雑な仕組みを短く命の危機へ落とす力がある。PRISMAでも、プリズム、属性、深淵、混沌を説明しすぎるより、プレイヤーの行動理由へ直結させる。

PRISMAへの適用:

- 「プリズムが奪われた」は、生活と命に直結させる。
- 「火が消えた」は、鍛冶だけでなく、飯、暖、包丁、農具に落とす。
- 「水が濁った」は、船、洗濯、魚、薬、宿の客足に落とす。
- 「深淵化」は、理屈よりも、知っている人が知っている声でなくなる怖さで見せる。

## 2. 複数視点レビュー

### 2-1. 小説読者の視点

評価:
素材は強い。特にジョセフとレナード、レイラとヴェルド、シャオとシャニー、ゼノンの闇の守護は、小説読者が追える感情線になっている。

問題:
一部イベントが「あらすじ」で済んでいる。キャラが見て、揺れて、選ぶ過程が薄い。レイラ即加入、ケイト同行、シャニー同行はその典型。

修正:
加入イベントは必ず「本人が見たもの」「まだ引っかかっているもの」「だから同行する小さな理由」を置く。

### 2-2. プロ編集者の視点

評価:
プロットの芯はよい。属性を思想化する構造は作品の武器。

問題:
場面役割が混ざる。戦闘後に、浄化、敵の捨て台詞、次目的地、加入、世界観説明を一気に処理しがち。

修正:
1イベントに残す感情を絞る。

例:
光宮殿勝利後は「ジャスパーの闇プリズム執着」「魔王軍誤認強化」「裂け目へ消える」で止める。レイラ加入は牢屋イベントへ分離する。

### 2-3. 中学生プレイヤーの視点

評価:
火→風→水→雷→光→闇→深淵という進行は分かりやすい。

問題:
抽象語や雰囲気語が入ると急に分からなくなる。「足りません」「術式の線」「現場で間に合わせます」は、意味より先に引っかかる。

修正:
難しい感情は、言葉を簡単にする。

悪い例:
「王国で聞かされてきた話だけでは、もう足りません」

良い方向:
「王国は正しい。そう信じてきました。でも、あの光は……人を救うものには見えませんでした」

### 2-4. ゲームをあまりプレイしない大人の視点

評価:
町NPCの生活感は入りやすい。鍛冶、洗濯、魚、鍋などはゲーム慣れしていない人にも伝わる。

問題:
ゲームのお約束である「勝ったから仲間になる」「ボスが目的地を言う」が唐突に見える可能性がある。

修正:
導線は残す。削らない。ただし、人間の事情に乗せる。

例:
ガルヴァニア洞窟北口を兵士に封鎖させ、レイラが必要だと自然に分からせる。

### 2-5. JRPGプレイヤーの視点

評価:
深淵の10階刻みボス構成は良い。30〜70階の因縁再戦は見せ場になる。

問題:
削る方向に行くと、ボス前会話が薄くなる。PRISMAは普段の会話量が多くないため、ボス前で濃く書く必要がある。

修正:
深淵は削らない。むしろ濃くする。ただし説明演説ではなく、敵の思想と味方の経験がぶつかる形にする。

## 3. 現在の最重要問題

### 3-1. キャラの根にない語が出る

NG例:

```js
{ "name": "ケイト", "text": "怖いです。指先が冷えると、術式の線まで震えます。", "charId": 104 }
{ "name": "ケイト", "text": "同行させてください。今度は、現場で間に合わせます。", "charId": 104 }
```

問題:
「術式の線」が何か不明。「現場で間に合わせます」は仕事人の報告のようで、ケイトの心ではない。

改善方針:
ケイトは、帽子・師匠の後ろ・声の小ささ・怖さ・負けず嫌いから言葉を出す。

例:

```js
{ "name": "ケイト", "text": "怖いです。今も、帽子を深くかぶってないと、顔を上げられません。", "charId": 104 }
{ "name": "ケイト", "text": "でも……この街の水が濁っていくのを、師匠の後ろから見てるだけなのは、もう嫌です。", "charId": 104 }
{ "name": "ケイト", "text": "行かせてください。僕の魔法で、少しでも水を戻せるなら……試したいんです。", "charId": 104 }
```

さらに日常会話へ寄せるなら:

```js
{ "name": "ケイト", "text": "怖いです。正直、外に出るより、机で本を読んでる方がずっと楽です。", "charId": 104 }
{ "name": "ケイト", "text": "でも、街の水がこんな色になってるのに、僕だけ部屋に戻るのは……嫌です。", "charId": 104 }
{ "name": "ケイト", "text": "行かせてください。師匠の弟子としてじゃなく、僕自身の頼みです。", "charId": 104 }
```

こちらの方が、造語がなく、少年の弱さと一歩出る感じがある。

### 3-2. 主人公の無言を他キャラが説明してしまう

NG例:

```js
{ "name": "ガイル", "text": "……今、行くって顔したよな。よし、俺も行く。", "charId": 109 }
```

問題:
主人公の意図を言語化し直していてしつこい。

改善方針:
主人公の頷きは頷きのままでよい。周囲は説明ではなく、覚悟や軽口で受ける。

例:

```js
{ "name": "システム", "text": "[N:301]は静かに頷いた。" }
{ "name": "ガイル", "text": "へへ……そうこなくっちゃな。", "charId": 109 }
{ "name": "サラ", "text": "ガイル。笑っていますけど、手、震えていますよ。", "charId": 110 }
{ "name": "ガイル", "text": "うるせえ。震えてても行くんだよ。", "charId": 109 }
```

### 3-3. 勇者様呼びの前提が弱い

問題:
ガイルやサラが最初から「勇者様」と呼ぶなら、村側にそれを受け入れる文脈がいる。

改善方針:
長老に短い伝承を置く。

例:

```js
{ "name": "長老", "text": "古い言い伝えがございます。世界が割れ、混沌が生まれし時……", "charId": 1001 }
{ "name": "長老", "text": "夜明けとともに、光を背負う勇者が現れる、と。", "charId": 1001 }
{ "name": "長老", "text": "まさか、この老いぼれが生きているうちに、その背を見ることになろうとは……。", "charId": 1001 }
```

ガイルは憧れ半分で乗る。
サラは恐縮と祈りで呼ぶ。

### 3-4. ソフィアがうざく見える危険

NG方向:

```js
{ "name": "ソフィア", "text": "だから頼むよ、命知らずの冒険者さんたち。こういう時に突っ込むのが仕事だろ？", "charId": 202 }
```

問題:
味方キャラなのに、読者が嫌いになる言い方。乱暴なだけで、弟子や街への責任が見えない。

改善方針:
ソフィアは癖のある人だが、弟子と街への情がある。依頼は「巻き込んですまないが、頼む」へ寄せる。

例:

```js
{ "name": "ソフィア", "text": "私は顔が知られすぎてる。表で動けば、宿にも水場にも兵が来る。", "charId": 202 }
{ "name": "ソフィア", "text": "巻き込んですまない。だが、あんたたちに頼ませてくれ。", "charId": 202 }
{ "name": "ソフィア", "text": "ケイト。怖いなら、怖いと言いな。……その上で、行けるかい？", "charId": 202 }
{ "name": "ケイト", "text": "怖いです。でも、行きます。街の水がこのままなのは、嫌です。", "charId": 104 }
{ "name": "ソフィア", "text": "よし。なら顔を上げな。帽子で隠れてても、足元は見えないよ。", "charId": 202 }
```

「帽子のつばが上がってる」より自然。帽子を小物として使うが、比喩を押しつけない。

### 3-5. レイラの疑念が抽象語になる

NG例:


改善方針:
レイラは、王国への信頼、父の変貌、光のプリズムへの違和感を、具体的に言う。

ただし、加入は光宮殿直後ではなく、牢屋会話で行う。

## 5. 深淵終盤の修正方針

深淵は削らない。密度を上げる。

ただし、次を守る。

- 伏線回収を説明しない。
- 敵の主張と味方の反論を論理的に噛ませる。
- 属性の定義を語るのではなく、属性の使い方を巡って対立させる。
- 因縁キャラがいる場合、その人の過去から言葉を出す。
- 任意加入キャラは追加会話。メイン感情線はストーリー加入キャラで成立させる。

### ヴェルド最期の主人公たちへの言葉

方針:
「救世主ごっこ」への対比を説明しない。ただ最後に、光の戻った目で「救世主たち」と呼ぶ。

提案:

```js
{ "name": "混沌の騎士ヴェルド", "text": "……世界を、守ってくれ。" },
{ "name": "混沌の騎士ヴェルド", "text": "救世主たち。" }
```

ここに「かつて嘲った」等は不要。

## 6. エピローグ方針

100階後は、一度完全にハッピーエンドにする。

- さらなる敵の匂わせは入れない。
- 属性ごとの演説にしない。
- 仲間同士の掛け合い、疲れ、飯、帰る場所、笑いを置く。
- 深淵のさらに奥は別イベントで開く。

提案の空気:

```js
{ "name": "ガイル", "text": "……終わった、んだよな？ なあ、終わったって言っていいんだよな？", "charId": 109 }
```

## 7. 最終修正方針の言語化

PRISMA ABYSSは、属性を集めるゲームではない。

火を、誰かを焼くために使う者と、誰かを温めるために灯す者の物語。
水を、世界を沈める力と見る者と、濁っても澄ませようとする者の物語。
風を、国の命令を運ぶ息と見る者と、子どもの背を押す風と見る者の物語。
雷を、最強の剣と見る者と、友の狂剣を止める盾と見る者の物語。
光を、選別と支配の白と見る者と、迷う足元を照らす灯と見る者の物語。
闇を、悪と呼ばれながら、世界の終わりを抱えて守った者の物語。

だから修正は、設定を増やすことではない。

キャラが、飯を食い、怖がり、怒り、間違え、好きな人を案じ、誰かの言葉を覚えていて、ボスの前でようやくそれを返す。

その連なりにする。

合言葉:

> 本筋は分かりやすく。  
> 日常は普通に。  
> ボス前は濃く。  
> 伏線は叫ばない。  
> キャラの言葉は、生活と傷から出す。
