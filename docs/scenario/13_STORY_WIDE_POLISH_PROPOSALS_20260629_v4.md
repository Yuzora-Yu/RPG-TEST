# 13_STORY_WIDE_POLISH_PROPOSALS_20260629_v4

Status: revised proposal  
Created: 2026-06-29  
Scope: PRISMA ABYSS story-wide polish proposal, after user review of v3.

## v4 policy update

v3の提案には、まだ以下の問題が残っていた。

- 口に根のない抽象語が残っていた。
- 生活感を足すつもりで、不自然な小道具や言い回しを出していた。
- 味方キャラの軽口が、不快・乱暴・責任転嫁に見える箇所があった。
- 本来画面演出で見せるべき場面を、テキストで説明しようとしていた。
- レイラ加入導線が、既存の想定と合っていなかった。

v4では、以下を優先する。

1. キャラの口から自然に出る日本語にする。
2. 目的地・次の行動は明確に伝える。
3. 誤認は弱めない。曖昧な否定を軽々しく入れない。
4. 味方キャラは癖があっても、読者に嫌われない範囲で描く。
5. 深淵ボス会話は削らない。終盤らしく濃くする。
6. 斬撃、消滅、登場、光の変化などは画面演出に逃がせるなら逃がす。

## Reference handling

ユーザー提示の参考作品は、本文模倣ではなく、会話設計の参考とする。

- 会話でキャラ同士の距離感を出す。
- シリアスの中にも、普通の言葉を混ぜる。
- 目的や本筋は見失わせない。
- 感情を説明しすぎないが、伝わらないほど削らない。
- 強い台詞の前後に、日常的な言葉や短い反応を置く。

詳細は `15_REFERENCE_DIALOGUE_STYLE_NOTES_20260629.md` を参照。

## S Priority: immediate story-logic corrections

### S-1. 光の宮殿勝利後イベント

#### Problem

- ジャスパー戦後にレイラが即加入するのは唐突。
- 魔王城へ向かう理由が弱い。
- ジャスパーの闇プリズムへの執着を使えば、魔王軍誤認を強めながら自然に次章へ進める。

#### Proposed flow

1. 光のプリズムが浄化される。
2. ジャスパーが闇のプリズムへの執着を漏らす。
3. 「忌まわしき魔王め。思い通りにはさせぬ……。」で、魔王が敵に聞こえる余地を残す。
4. ジャスパーとヴェルドが裂け目へ消える。
5. シャオが魔王軍誤認を強める。
6. 一度、宮殿の牢屋へ戻る目的を出す。
7. 牢屋でレイラと会話し、レイラが自分の意志で加入する。

#### Approved Jasper / Xiao core lines

```js
{ "name": "魔道神官ジャスパー", "text": "…真の光には闇が…闇のプリズムさえあれば、まだ……。", "charId": 501 },
{ "name": "魔道神官ジャスパー", "text": "忌まわしき魔王め。思い通りにはさせぬ……。", "charId": 501 },

{ "name": "シャオ", "text": "闇のプリズムまで失ったら終わりだ。行こう、魔王城へ。", "charId": 105 }
```

## A Priority: dialogue correction policies

### A-1. ソフィアは乱暴にしない

#### Rejected direction

```js
{ "name": "ソフィア", "text": "だから頼むよ、命知らずの冒険者さんたち。こういう時に突っ込むのが仕事だろ？", "charId": 202 }
```

#### Why rejected

- 読者がソフィアを嫌いになる。
- 仲間を危険へ押し出す言い方に見える。
- ソフィアの癖を出そうとして、味方としての信頼感を損なっている。

#### Correct direction

```js
{ "name": "ソフィア", "text": "私はこの街で顔を知られすぎている。\n表で動けば、宿にも市場にも兵が来る。", "charId": 202 },
{ "name": "ソフィア", "text": "巻き込んですまない。\nけれど、今この街で動けるのは、あんたたちだけなんだ。", "charId": 202 }
```

ソフィアは皮肉屋で色気もあるが、味方に対して責任を押し付ける人物ではない。

### A-2. ケイトは日常会話に寄せる

#### Rejected direction

```js
{ "name": "ケイト", "text": "怖いです。指先が冷えると、術式の線まで震えます。", "charId": 104 }
{ "name": "ケイト", "text": "また誰かが沈むのを見てるだけになりますから。", "charId": 104 }
```

#### Why rejected

- `術式の線` が唐突。
- `沈むのを見てるだけ` は重く作りすぎ。
- ケイトの普通の怯え、師の後ろに隠れてきた過去、今ここで一歩出たい気持ちが自然に出ていない。

#### Correct direction

```js
{ "name": "ケイト", "text": "し、師匠。僕も……行きます。", "charId": 104 },
{ "name": "ソフィア", "text": "怖いなら残りな。\n無理に前へ出ろとは言わないよ。", "charId": 202 },
{ "name": "ケイト", "text": "怖いです。でも……ここで残ったら、\n僕はまた、師匠の後ろに隠れるだけです。", "charId": 104 },
{ "name": "ケイト", "text": "僕の魔法で足りるなら、\n水上都市の人たちを助けたいです。", "charId": 104 }
```

### A-3. レイラの表現を自然にする

#### Rejected direction

```js
{ "name": "レイラ", "text": "王国で聞かされてきた話だけでは、もう足りません。", "charId": 204 }
```

#### Why rejected

- `足りません` が抽象的で、何が不足しているのか分からない。
- 王国がおかしい、父が変わった、プリズムの力が何に使われているのか、という本筋がぼやける。

#### Correct direction

```js
{ "name": "レイラ", "text": "王国は正しい。ずっと、そう教わってきました。", "charId": 204 },
{ "name": "レイラ", "text": "でも、宮殿で見た光は……人を救うものには見えなかった。", "charId": 204 },
{ "name": "レイラ", "text": "ヴェルド様がなぜああなったのか。\nプリズムの力を、誰が何に使っているのか。", "charId": 204 },
{ "name": "レイラ", "text": "魔王城で確かめます。\n私の剣を、もう誰かの言葉だけに預けたくありません。", "charId": 204 }
```

### A-4. 魔王軍誤認は弱めない

#### Policy

- シャオは魔王軍を疑い、魔王を倒すべきだと考えてよい。
- この段階でケイトやレイラに曖昧な否定を入れると、読者へ真相を教えてしまう。
- 断定に対する曖昧な否定は、多くの場合「否定側が正しい」という読者サインになる。
- したがって、魔王城前までは誤認を弱めすぎない。

### A-5. THUNDER_LEONARD_CLEAR

#### Policy

- レナードが斬られて消えるシーン、騎士が登場するシーンは、台詞説明より画面演出で見せる。
- テキストでは、ジョセフの反応、レナードの短い言葉、騎士たちの存在感だけを置く。
- ボス戦前の台詞は削らない。因縁の場面として必要。

## B Priority: tone and structure

### B-1. 始まりの村の勇者様呼び

`勇者様` 呼びは、主人公の職業が勇者である以上、採用してよい。  
ただし、唐突に見えないよう、村に古い伝承を置く。

```js
{ "name": "長老", "text": "古い言い伝えがございます。\n世界が割れ、混沌が生まれし時……", "charId": 1001 },
{ "name": "長老", "text": "夜明けとともに、光を背負う勇者が現れる、と。", "charId": 1001 },
{ "name": "長老", "text": "まさに、伝承の勇者様のようでしたぞ。", "charId": 1001 }
```

その後に、ガイルやサラが少しずつ `勇者様` と呼ぶ流れにすれば自然。

### B-2. 主人公の無言を他キャラに説明させない

#### Rejected direction

```js
{ "name": "ガイル", "text": "……今、行くって顔したよな。よし、俺も行く。", "charId": 109 }
```

#### Why rejected

- 主人公の意志を別キャラが説明し直している。
- しつこい。
- 元の `静かに頷いた` の方がまだ自然。

#### Correct approach

- 主人公の無言は無言として扱う。
- 反応させるなら、説明ではなく、場の温度やキャラの感情で受ける。

```js
{ "name": "ガイル", "text": "へへ……そうこなくっちゃな。", "charId": 109 },
{ "name": "サラ", "text": "ガイル。笑ってますけど、手、震えてますよ。", "charId": 110 },
{ "name": "ガイル", "text": "うるせえ。震えてても行くんだよ。", "charId": 109 }
```

## C Priority: deep abyss handling

### C-1. 深淵ボス会話は削らない

- 深淵はクライマックス。
- 普段の会話シーンが少ないゲームだからこそ、ボス前会話は濃密でよい。
- 各属性の歪み、仲間の反論、過去の因縁をしっかり置く。
- ただし説明演説にならないよう、相手の主張と味方の反論を噛み合わせる。

### C-2. 100階後エピローグ

- 属性ごとの演説にしない。
- 仲間同士の掛け合い、疲労、安堵、帰る場所、飯、笑いを使って一時の大団円にする。
- さらなる敵の匂わせはこの場では不要。
- 完全なハッピーエンドに見せた後、別イベントで深淵のさらに奥を開く。

## D Priority: character sheet work

### D-1. 好きなもの・嫌いなもの

属性記号に寄せすぎない。  
自然な生活・過去・性格に根差したものを書く。

悪い方向:

- 火属性だから辛いものが好き
- 水属性だから雨が好き
- 光属性だから白い花が好き

良い方向:

- ガイル: 腹にたまる飯、古い英雄譚、旅の記録を書く時間
- サラ: 薬草を干す匂い、子どもが安心して眠る顔、温かいスープ
- ケイト: 静かな机、師匠の古いメモ、誰も急かさない時間
- レイラ: 朝の鍛錬後に飲む水、手入れされた装備、リンとの噂話
- シャニー: 古い詩集、夜明け前の静けさ、妹の話を誰にも聞かれない時間

### D-2. キャラ詳細シートを先に作る

台詞を直す前に、以下を作る。

- 性格
- 過去
- 好きなもの
- 嫌いなもの
- 苦手なもの
- 口癖
- 特技
- 他キャラとの距離感
- 言いそうな言葉
- 言わせない言葉

## Final note

v4以降は、単に“それっぽい良い台詞”を作るのではなく、以下を毎回確認する。

> 誰が、誰に、何を受けて、何のために言ったのか。

これが答えられない台詞は、どれだけ綺麗でも不採用。
