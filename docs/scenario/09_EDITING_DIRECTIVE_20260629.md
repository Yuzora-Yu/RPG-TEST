# 09_EDITING_DIRECTIVE_20260629

2026-06-29時点の編集方針。

## Overall diagnosis

物語の骨格は強い。

- 王道RPGとしてプリズムを巡る旅が分かりやすい。
- 王国軍が単純な正義ではない。
- 魔王軍も単純な悪ではない。
- 最後に敵意の焦点が深淵へ移る流れが気持ちよい。
- シャオの魔王軍への誤認が、本人の傷と結びついている。

一方で、AIが足したと思われる町人会話・中継イベント・一部加入イベントは温度が落ちる。

弱い文の特徴:

- 設定資料を短く分割して喋る。
- 全員が同じ詩的な調子になる。
- 町人が世界のために喋りすぎる。
- プレイヤーが疑う前に答えを出す。
- 敵が怖くなる前に事情を匂わせる。
- `Aは、Bだ。Cは、Dだ。` が連続する。

## Priority

既存イベントを時系列で保ったまま、AIっぽい台詞を人間の会話に戻す。

優先順位:

1. 村人・町人会話
2. 仲間加入前後の会話
3. 主人公発話の削除・置換
4. エルメナスの男魔族化
5. 魔王城の戦闘前後の順番修正
6. レイラの高潔さと感謝の補強
7. シャオの誤解と感情の長期的な積み上げ

## Town NPC rule

町人は攻略情報より、世界に戻ってきたいと思わせる空気を作る役。

町人が話してよいもの:

- 自分の畑
- 自分の子供
- 自分の商売
- 自分の怖さ
- 自分の腹立ち
- 近所の噂
- 見間違い
- 信じている嘘
- 二度目にだけ出る本音

避けるもの:

- 世界の真相
- プリズムの仕組み
- 王国軍の真相の断定
- 魔王軍の真相の断定
- 次の目的地だけを言う台詞

## Before and after examples

弱い方向:

```text
黒い風は消えた。
鳥は、まだ戻ってこない。
```

良い方向:

```text
「森に近づくなよ」
「昨日、うちの父ちゃんが
　帰ってこなかったんだ」
```

弱い方向:

```text
北東の穴は、昨日まで
ただの草地だったんだ。
```

良い方向:

```text
「そこ、踏むなよ」
「昨日まで、うちの草地だった」
「柵も鍬も飲まれた」
「……次は家かもしれねえ」
```

## Spoiler pacing

プレイヤーが疑う前に、文章が答えを出してはいけない。

序盤から避ける:

- 実は魔王は良い奴かも
- 王国軍こそ怪しい
- プリズムは本当はこう
- 深淵の王が真の敵

代わりに置く:

- 兵士の態度のズレ
- 子どもだけが見たもの
- 町の噂の食い違い
- 事件後も戻らない生活
- 感謝と怒りが同居する会話

## Character direction

### シャオ

- 魔王軍が悪いに決まっていると思いたがる。
- 王国軍への疑念に早く乗りすぎない。
- シャニーの前でだけ弱くなる。
- 誤解は欠点ではなく、傷が作った視界として扱う。

### レイラ

- 高潔な聖騎士。
- 弱っていても礼を失わない。
- 助けられたらまず感謝する。
- 王国を信じたい気持ちと、目の前の民を救う責任の間で揺れる。
- ヴェルドは幼いレイラを拾い、育てた後見人/養父。
- レオンは騎士としての正式な師。
- ヴェルドは家族、レオンは騎士道の理想として分ける。
- 深淵70階ヴェルド戦で「家族への礼節」と「娘としての叫び」を両方出す。

### ゼノン

- 戦闘前に善性を見せない。
- 言い訳しない。
- 怖い王として立つ。
- 戦闘後、闇のプリズムに異常がないことで「守っていたのか」と分かる。
- 善人ではなく、約束を破らない王。

## Abyss and chaos canon

混沌は第七属性ではない。

正しい扱い:

- 深淵は六属性の理が割れた傷。
- 王国のプリズム崩壊の儀により、六属性の調和に裂け目が生まれた。
- その裂け目から滲み出した、属性ならざる歪みが混沌と呼ばれた。
- アゼルガラグはその歪みから生まれた影/王を名乗る存在。

使ってよい誤解:

- ジャスパーだけが混沌を「第七属性」と呼ぶ。
- ただし、それは誤った研究者の分類として扱う。
- ジャスパーの傲慢さと破滅を示す言葉にする。

100階後の扱い:

- アゼルガラグ撃破で、奪われていた属性の残滓は解放される。
- 地上へ漏れ出していた深淵化は鎮まり、世界は一度息を吹き返す。
- しかし深淵そのものは消えない。
- 王を名乗ったそれすら、混沌の傷から生まれた影にすぎなかった。
