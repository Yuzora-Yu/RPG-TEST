# 02_CHARACTER_VOICE_BIBLE

キャラクターごとの口調・感情・誤解・成長を管理する。

このファイルは、既存 `characters.js` と `docs/story-bible/` を読んだ上で更新する。

## Voice policy

全キャラが同じ話し方になることを禁止する。

ラノベ的なくらい、話者ごとの癖を出してよい。
ただし、過剰な語尾記号だけで個性を作らない。

## Special focus: Xiao

シャオはミスリードを担える重要キャラ。

### Direction

- 魔王軍への恨みや疑いを持ちやすい。
- シャニー関連の感情が判断を曇らせてよい。
- 王国軍の異常より、魔王軍の暗躍を疑いやすい。
- ただし、単なる短絡キャラにしない。
- 自分の誤解に気づく過程を成長として描く。

### Bad usage

- ただ怒鳴るだけ。
- 作者のミスリード装置として不自然に疑う。
- 読者が「明らかに違う」と感じるほど露骨に外す。

### Good usage

- 失ったものへの感情から、見えるものが歪む。
- 目の前の小さな違和感を見落とす。
- 後に、見落とした相手の言葉を思い出す。

## ジャスパー

### Basic
- charId: 未整理
- 初出: 光の神殿 / 深淵90階
- 役割: 王国崩壊の戦犯、歪んだ光の価値観を持つ神官
- 一人称: 儂
- 二人称: 貴様 / 愚者 / 迷える子ら
- 基本語尾: じゃ、ぬ、よ、である調を混ぜる
- 口調の硬さ: 非常に高い
- 話す速度: 遅く、断定的。興奮時は同じ語を重ねる
- 感情温度: 冷たい傲慢。追い詰められると幼い執着が漏れる

### Vocabulary
- よく使う語: 光、導き、選別、分類、成果、式、真理、支配
- 避ける語: 闇に堕ちた、破滅した、悪そのもの
- 怒った時の語彙: 愚者、黙れ、選ばれぬ者、民に選ばせるな
- 怖い時の語彙: 王よ、まだ、儂は、成果、導ける

### Behavior in dialogue
- 悪事を悪事として語らない。救済・導き・光の語彙で正当化する。
- 人の顔を見ない。暮らしを数、分類、成果へ置き換える。
- 混沌を「第七属性」と誤認し、名づければ支配できると思っている。
- 敗北後も反省しない。最後まで「自分は照らす側だ」と信じる。

### Sample lines
- 通常: 「民に選ばせるから過つのだ。」
- 傲慢: 「儂が照らし、儂が選び、儂が世界を正す。」
- 追い詰められた時: 「違う。儂は器ではない。儂は選ばれた者だ。」
- 味方からの反論軸: 「あなたの光は、人を照らしていない。ただ、目を焼いているだけです。」

## Voice review checklist

各シーンで確認すること:

- この話者でなくても言えるセリフになっていないか。
- 全員が同じ説明役になっていないか。
- 感情が抽象説明になっていないか。
- 口調の個性が語尾だけになっていないか。
- その時点の誤解や無知が反映されているか。
- 長文を1回で話していないか。
