# 11_TOWN_DIALOGUE_REWRITE_BATCH_20260629

Status: draft

既存の町人会話を、設定説明ではなく生活者の声へ寄せるための改稿バッチ。

この文書は実装前ドラフト。`story.js` / `map.js` へは未反映。

## Rewrite principles

- 町人は世界の真相を知らない。
- 町人は自分の生活から話す。
- 王国軍や魔王軍への評価は、噂・怖さ・利害で揺らす。
- 詩的な一文より、手触りのある困りごとを優先する。
- 同じ町でも、信じている噂が食い違ってよい。

## Start village

### Villager: 畑の持ち主

Timing: 洞窟調査前

Purpose:

- 大穴や魔物の異変を生活被害として見せる。
- 世界説明ではなく、畑と家への恐怖で始める。

```text
村人：
「そこ、踏むなよ」
「昨日まで、うちの草地だった」

村人：
「柵も鍬も飲まれた」
「……次は家かもしれねえ」
```

After cave boss:

```text
村人：
「穴は小さくなった」
「でも、牛がまだ戻らねえ」

村人：
「助かったよ」
「礼は……芋でいいか？」
```

## Implementation notes

- 既存NPC会話への反映は未承認。
- 実装前に、各町の現行NPCソース位置を特定する。
- 同じ町で全員が同じ調子にならないよう、役割ごとに口調を変える。
- 1回目/クリア後/二度目会話の差分を持たせると効果が高い。
