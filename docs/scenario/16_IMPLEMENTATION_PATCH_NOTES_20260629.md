## 1. 光の宮殿勝利後イベント

### Current issue

- ジャスパー戦後、レイラが即加入する流れが唐突。

- ジャスパーの台詞から魔王城への導線を作り、シャオが魔王軍誤認を強める方が自然。

### Correct direction

- ジャスパーは闇のプリズムへの執着を口にする。
- 「魔王」が敵に聞こえる台詞を残す。
- ジャスパーとヴェルドは次元の裂け目へ飲み込まれる。
- シャオが魔王軍への疑念を確信に近づける。

## 4. ソフィア・ケイト 水上都市加入会話

### Current problem

過去案のソフィアが乱暴すぎる。  
`こういう時に突っ込むのが仕事だろ？` は、味方キャラとして読者に嫌われる危険が高い。

### Correct direction

- ソフィアは癖があるが、非情ではない。
- 自分が動けない理由は、街で顔が知られており、宿や関係者に迷惑がかかるため。
- 冒険者たちを利用するのではなく、巻き込んでしまうことに申し訳なさを見せる。
- ケイトは気弱だが、師の影に隠れ続けてきた自分を変えたい。
- ケイトの才能は、鍾乳洞や海底神殿で画面演出・戦闘・短い反応によって片鱗を見せる。

### Proposed dialogue

```js
{ "name": "ソフィア", "text": "私はこの街で顔を知られすぎている。\n表で動けば、宿にも市場にも兵が来る。", "charId": 202 },
{ "name": "ソフィア", "text": "巻き込んですまない。\nけれど、今この街で動けるのは、あんたたちだけなんだ。", "charId": 202 },
{ "name": "ケイト", "text": "し、師匠。僕も……行きます。", "charId": 104 },
{ "name": "ソフィア", "text": "怖いなら残りな。\n無理に前へ出ろとは言わないよ。", "charId": 202 },
{ "name": "ケイト", "text": "怖いです。でも……ここで残ったら、\n僕はまた、師匠の後ろに隠れるだけです。", "charId": 104 },
{ "name": "ケイト", "text": "僕の魔法で足りるなら、\n水上都市の人たちを助けたいです。", "charId": 104 },
{ "name": "ソフィア", "text": "……言うようになったじゃないか。\nなら、行っておいで。帰ったら叱ってやる。", "charId": 202 }
```

### Notes

- `帽子のつば、今日はずいぶん上がってるじゃないか。` はまだ作り物感があるため不採用。
- `また誰かが沈むのを見てるだけになりますから。` も重すぎて不自然。日常会話に寄せる。

- 深淵ボス会話は削らない。
