## 実機確認

- 氷床6フロアは自動検証済み。スマートフォン実機で、長押し入力時の停止点と隣接操作ボタンの表示を確認する。
- 最終階カーペットについて、NPC・ボス・イベントが上にいる状態で織物が欠けないことを複数画面比率で確認する。
- キャッシュ確認で「いいえ」を選んだ状態でも、追加した操作盤・ござ・カーペット素材がバックグラウンド取得対象になることを継続確認する。

## アセット保守

- 画像には互換キー、ランタイム用、`managed/source` の原本があり、同一内容のファイルが複数存在する。参照グラフを確認せず重複画像を削除しない。
- 重複整理を行う場合は、`assets.js`、各ライブラリの `manifest.json`、HTML/CSS/JS参照、Service Workerの全キャッシュ列挙を同時に検査する。
- 生成済みモンスターは、深層雑魚・既存クエストボス・ハンターへの採用後も戦闘バランスと図鑑表示を実機確認する。
